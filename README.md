# india-literacy-analysis
My beginner project for learning purpose on real world datasets.

## Project Overview
This is a beginner data analysis project using India's 2011 Census dataset
to understand real world problems and data's patterns.

## Analysis Performed
- Total literacy rate
- Urban vs Rural comparison on literacy
- Male vs Female comparison on literacy

## Key Insights
- Kerala has the highest literacy rate whereas bihar seems to be the lowest
- Urban literacy rate is higher than rural areas
- Male literacy rate is higher than female literacy
- There is a noticeable literacy gap in rural regions

##  Tools Used
- Python
- Pandas
- Matplotlib
- seaborn

## Dataset
2011 Census data (used for learning purposes)

## Conclusion
This project helped me understand data cleaning, analysis, and visualization using real-world data.
